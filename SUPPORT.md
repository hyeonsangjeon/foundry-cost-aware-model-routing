# Support

This is a community sample accelerator maintained on a best-effort basis. It is
**not** an official Microsoft / Foundry product and carries no SLA or support
guarantee.

## Getting help

- **Questions / ideas:** open a GitHub Discussion or Issue on this repository.
- **Bugs:** open an Issue with steps to reproduce against a clean checkout
  (`git clone` → `make check` / `make replay`).
- **Security problems:** follow [`SECURITY.md`](SECURITY.md) — do not file a
  public issue.

## Before filing

- Confirm the problem reproduces from a clean checkout with the offline path
  (no Azure required).
- Include your OS, Python version, and the exact command and output.
- Never paste secrets, tenant identifiers, or raw live responses into an issue.
